# a-Go

è¶…A&G recording with RTMPdump from Go(exec)

# Use

Get

```
git clone
```

Arrangement

* rtmpdump.exe or rtmpdump install(yum,apt-get...etc)
* setting.json
  * recTime(sec,default:3900)
  * output dir(default:dist)
  * rtmp url(default:AG)

output directory

```
mkdir dist
```

Go Command Run

```
go run main.go

or

go build main.go
```

Operation check

* Windows 10 (Task Runner)
